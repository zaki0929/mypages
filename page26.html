Chrome のスーパーリロードでキャッシュを無視する

Chrome の スーパーリロードを使えば、キャッシュを無視してページをリロードすることができます。

これをつかうと、js や css の変更を即座に確認できるため、大変便利です。

このページではスーパーリロードのショートカットキーを紹介します。

ショートカットキー

Mac

Cmd + Shift + r

Windows

Shift + F5

ブラウザのタブやツールバーを非表示にしてフルスクリーン表示する

開いてるページの部分のみをフルスクリーン表示したい機会があったため、方法を調べました。

このページでは、それをするためのショートカットキーを紹介します。

動作を確認したブラウザ

  • Google Chrome (Windows, Ubuntu, Mac で確認)
  • Mozilla Firefox (Windows, Ubuntu で確認)
  • Microsoft Edge (Windows でのみ確認)
  • Internet Explorer (Windows でのみ確認)

ショートカットキー

Windows

F11

Ubuntu

F11

Mac

Cmd + Shift + F

*Mac の場合は全画面表示にしてからショートカットキーを入力してください。

突然 Invalid editor window エラーがでた - Unity

Unity の Editor Window 周りの開発をしているときに、突然 Invalid editor window エラーがでたので解決方法をメモします。

使用環境

Unity 2019.2.13f1

症状

突然、以下のようなエラーログが出力されるようになりました。

Invalid editor window  UnityEditor.EditorApplicationLayout:FinalizePlaymodeLayout()

解決方法

Unity 上部のメニューから以下を選択し、レイアウト設定を工場出荷状態に戻すことで、解決しました。

Window > Layouts > Revert To Factory Settings...

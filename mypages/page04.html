便利なエイリアス

このページでは Linux 環境で私が使用している、便利なエイリアスを紹介します。
以下で紹介するものを .bashrc に書き加えることで使用することができます。

どこでも catkin_make

これを書いておくとどこでもcatkin_makeできるようになります。
しかし、catkin_make --pkg などオプションを使う場合は他に工夫が必要になります...。

alias catkin_make='cd ~/catkin_ws && catkin_make && cd -'

らくらく nmap

これを書いておくと、すぐに nmap を使ってネットワーク上のホスト名と、アドレスをあぶり出すことができます。
ssh で繋ぐ前などに使うと便利です。

研究室で使う場合を勝手に想定して名前を nmaplab にしておきます。

IP address : xxx.xx.xx.ooo

alias nmaplab='nmap -sL xxx.xx.xx.0/24'

CakePHP3 でフォームのボタンに ID をつけたい

CakePHP3 でフォームのボタンに ID をつけたい機会があったので、その方法を書き残しておきます。

環境

  • CakePHP 3.6.10

方法

以下のようなフォーマットで記述するとID をつけたボタンが作成できます。

$this->Form->button('text', ['id' => 'button-id'])

例えば、OK という文字を入れたボタンに submit-button という ID をもたせたいときは以下のように記述します。

$this->Form->button('OK', ['id' => 'submit-button'])

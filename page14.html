Ubuntu 16.04 に CakePHP3 をインストールするメモ

最近 CakePHP を使う機会があったため、ここにメモを残しておきます。

このページでは Ubuntu 16.04 で CakePHP3 をインストールしてプロジェクトを作成するところまでの手順を紹介します。

環境

  • Ubuntu 16.04

インストール

apt で以下のようにインストールしていきます。

一行で書いたら見えにくかったので、一つずつ書きました。

$ sudo apt install php7.0
$ sudo apt install php7.0-intl
$ sudo apt install php7.0-mbstring
$ sudo apt install php7.0-xml
$ sudo apt install composer

Composer というのは PHP のパッケージ管理システムです、今回はこの Composer を使って CakePHP3 のプロジェクトを作成します。

プロジェクトの作成

今回は hellocakeという名前のプロジェクトを作ってみます。

まずは以下のコマンドを実行してプロジェクトを作成します。

$ composer create-project --prefer-dist cakephp/app hellocake

続いて必要なライブラリをインストールしましょう。

以下のように、上のコマンドで作成されたディレクトリに移動し、composer install を実行します。

$ cd hellocake
$ composer install

以上でプロジェクトの作成は完了です。

試しにサーバーをたててみる

プロジェクトを作成してみたので試しにサーバーをたててみましょう。

以下のコマンドで開発用のサーバーをたてることができます。

$ bin/cake server

サーバーがたったらブラウザで以下のアドレスにアクセスしてみましょう。

http://localhost:8765/

Database の欄に不穏な雰囲気がみられるのは、まだ Database の接続設定をしていないためなので問題ないです。

Database を扱いたいときには適宜設定をしてください。

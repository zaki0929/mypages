443ポートで git

sshポートが開いていない環境では、git で clone や push するとき不便です。
以下の設定で443ポートから git が使えるようになります。

環境

  • Ubuntu 16.04 server
  • Raspberry Pi 3 model B

    • 解決方法

    $ vim ~/.ssh/config
    Host github.com 
    HostName ssh.github.com 
    Port 443
    IdentityFile ~/.ssh/id_rsa

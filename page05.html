443ポートで git

ssh ポートが開いていない環境では、git で clone や push するとき不便です。
以下の方法で設定をすると443ポートから git が使えるようになります。

環境

  • Ubuntu 16.04

方法

$ vim ~/.ssh/config
Host github.com
    HostName ssh.github.com
    Port 443
    IdentityFile ~/.ssh/id_rsa

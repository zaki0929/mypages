CakePHP3 で composer install しようとしたら "The following exception is caused by a lack of memory and not having swap configured" といわれた

CakePHP3 で composer install しようとしたら "The following exception is caused by a lack of memory and not having swap configured" といわれたのでこれを解決した方法を残しておきます。

これは Composer のエラーなので、CakePHP3 以外の用途でこのエラーと遭遇した方でも解決するかもしれません 。

環境

  • Ubuntu 16.04

解決した方法

以下のようにして、swap 領域を作ってあげることで解決しました。

$ sudo /bin/dd if=/dev/zero of=/var/swap.1 bs=1M count=1024
$ sudo /sbin/mkswap /var/swap.1
$ sudo /sbin/swapon /var/swap.1
